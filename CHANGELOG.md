# Changelog

## 1.0.0 — Initial public release

- Modular McOS system layout and standalone installer
- Desktop, Start menu, taskbar, touch-monitor support, and startup guide
- Advanced file manager with Recycle Bin
- McNet networking, messaging, file transfer, ping, and remote-control permissions
- Redstone automation, peripheral management, notifications, logs, backup, and recovery
- Built-in utilities and external application support through McStore
- Reliability and safety audit covering installation, storage, networking, UI, and recovery paths
